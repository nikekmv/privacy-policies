## 🇺🇦 Українська

### 1. Загальні положення

Ця політика конфіденційності описує, які дані збирає додаток Garden Map і як вони використовуються.

### 2. Які дані збирає додаток

Garden Map не збирає, не передає і не зберігає жодних персональних даних на зовнішніх серверах.

Всі дані, які створює користувач, зберігаються **виключно локально на пристрої**:
- схема садової ділянки (зони, об'єкти, доріжки)
- інформація про рослини (назва, дата посадки, розміри, нотатки)
- фотографії рослин та записів щоденника
- записи садового щоденника
- завдання та нагадування
- налаштування додатку

### 3. Дозволи додатку

Garden Map може запитувати такі дозволи:

- **Камера** — для фотографування рослин та записів щоденника
- **Сховище / Медіафайли** — для вибору фото з галереї та збереження даних
- **Сповіщення** — для нагадувань про заплановані завдання (полив, підживлення тощо)

Жоден з цих дозволів не використовується для збору або передачі даних третім особам.

### 4. Передача даних третім особам

Garden Map **не передає** жодних даних третім особам, рекламним мережам або аналітичним сервісам.

Функція «Поради від ШІ» формує текстовий запит на основі вмісту вашої ділянки та копіює його в буфер обміну. Користувач самостійно вирішує, куди вставити цей запит (ChatGPT, Claude тощо). Додаток не надсилає жодних запитів до зовнішніх ШІ-сервісів автоматично.

### 5. Монетизація

Garden Map використовує Google Play Billing для обробки одноразової покупки розблокування повного доступу. Платіжні дані обробляються виключно Google — додаток не має доступу до платіжної інформації користувача.

### 6. Безпека даних

Оскільки всі дані зберігаються локально на пристрої, їх безпека залежить від налаштувань безпеки самого пристрою.

### 7. Зміни до цієї політики

У разі оновлення політики нова версія буде опублікована за цим посиланням. Рекомендуємо періодично переглядати цю сторінку.

## 🇬🇧 English

### 1. General

This Privacy Policy describes what data the Garden Map app collects and how it is used.

### 2. Data Collection

Garden Map does not collect, transmit, or store any personal data on external servers.

All data created by the user is stored **exclusively on the local device**:
- garden layout (zones, objects, paths)
- plant information (name, planting date, size, notes)
- photos of plants and diary entries
- garden diary records
- tasks and reminders
- app settings

### 3. App Permissions

Garden Map may request the following permissions:

- **Camera** — to take photos of plants and diary entries
- **Storage / Media** — to select photos from the gallery and save app data
- **Notifications** — to send reminders for scheduled tasks (watering, fertilizing, etc.)

None of these permissions are used to collect or share data with third parties.

### 4. Third-Party Data Sharing

Garden Map **does not share** any data with third parties, advertising networks, or analytics services.

The "AI Advice" feature generates a text prompt based on your garden content and copies it to the clipboard. The user decides where to paste it (ChatGPT, Claude, etc.). The app does not automatically send any requests to external AI services.

### 5. Monetization

Garden Map uses Google Play Billing to process a one-time purchase for unlocking full access. Payment data is processed exclusively by Google — the app does not have access to the user's payment information.

### 6. Data Security

Since all data is stored locally on the device, its security depends on the security settings of the user's device.

If this policy is updated, the new version will be published at this link. We recommend reviewing this page periodically.
